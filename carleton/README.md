# Chelsea's Kitchen — Thursday Lunch Club

Upload-ready, responsive GitHub Pages order page for $15 Thursday lunch delivery at Carleton University.

## Upload

1. Open the `ChelseasKitchen` repository.
2. Create or replace the `carleton-lunch` folder.
3. Upload `index.html`, `styles.css`, `script.js`, and the complete `assets` folder.
4. Commit the changes.
5. Open `https://chelseaskitchen.ca/carleton-lunch/`.

Orders are prepared as pre-addressed emails to `kris@chelseaskitchen.ca`. Payment remains Square tap or cash on delivery.
